; Enable PHP GD extension in php.ini
; Add this line to your php.ini file:

extension=gd

; Common php.ini locations:
; Windows: C:\tools\php82\php.ini
; Or: C:\xampp\php\php.ini
; Or: C:\wamp\bin\php\php8.2.x\php.ini

; Steps:
; 1. Find your php.ini file
; 2. Search for "extension=gd" - if commented out with ";", uncomment it
; 3. If not found, add "extension=gd" to the dynamic extensions section
; 4. Restart PHP server
; 5. Verify by visiting: http://localhost:8000/api/test-qr-print.php
;    Should now show "gd_available": true
